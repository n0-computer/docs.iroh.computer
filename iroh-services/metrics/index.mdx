When you add endpoints to your network, various metrics are collected to help improve
performance and user experience. 

Endpoints added to your network can opt-in to send these metrics to the Iroh
Services platform which will appear on the project dashboard.

## Project metrics 

All endpoint metrics are aggregated at the project level, to provide insights
into overall network performance. This aggregated data helps in understanding
trends and patterns without exposing individual endpoint details.  

Read more about [project metrics](/iroh-services/metrics/index). 

## Endpoint metrics

Mmore detailed metrics are available for each endpoint added to the network.
These metrics are available at the endpoint level, so you can drill down into
specific behaviors to better understand performance issues. Projects that have
been upgraded to Enterprise plan have access to endpoint-level
metrics retention and lookback.

Read more about [endpoint metrics](/iroh-services/metrics/endpoint). 


## Privacy

All metrics collected are anonymized and aggregated to ensure user privacy.
No personally identifiable information (PII) is collected or stored. The
focus is on overall network performance rather than individual user behavior.

If you have concerns about data collection, please refer to the
[privacy policy](https://n0des.iroh.computer/legal/privacy).
